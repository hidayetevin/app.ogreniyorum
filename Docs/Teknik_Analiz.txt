# Projenin Amacı
4–10 yaş arası çocuklar için web tabanlı, güvenli ve erişilebilir bir resim eşleştirme oyunu geliştirmek. Amaç, görsel hafıza, dikkat ve eşleştirme becerilerini eğlenceli bir deneyimle desteklemek, ebeveyn dostu kontrol seçenekleriyle reklam ve satın alma davranışlarını şeffaf biçimde yönetmektir.

# Teknik Detaylar
- Dil: JavaScript veya TypeScript (tercih: TypeScript)
- Oyun motoru: Phaser.js (v3.60+)
- Araçlar: Vite, ESLint, Prettier
- Stil: CSS (erişilebilirlik odaklı)
- Medya: Sprite atlasları, ses dosyaları, JSON konfigürasyon
- Yerelleştirme: JSON sözlükleri (tr, en)
- Depolama: LocalStorage
- Analitik: Minimal, anonim olay kaydı
- Monetizasyon: Reklam entegrasyonu için soyut katman

# Proje Yapısı
- src/assets: Görseller, sesler
- src/config: Kategori ve seviye JSON’ları
- src/core: Servisler (Audio, Feedback, Storage)
- src/scenes: MainMenu, CategorySelection, GamePlay, LevelComplete
- src/ui: HTML overlay’leri (ebeveyn paneli, ayarlar)
- tests: Birim testleri

# Oyun Tasarımı
- Durum makinesi: MainMenu → CategorySelection → GamePlay → LevelComplete
- Veri modelleri: Kategori, Seviye, Kart, Ayarlar, İlerleme
- Servisler: FeedbackService, AudioService, StorageService, LevelService, AnalyticsService
